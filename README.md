# arca-galactica
Videojuego de acción espacial en Java con arquitectura MVC. Los alienígenas están abduciendo animales de toda la galaxia. Pilota el Arca Galáctica, destruye los ovnis y rescátalos antes de que desaparezcan para siempre. Autores:  Juan Miguel Osorio López Juan Daniel Pineda Pineda Juan José Ospina Guzmán
